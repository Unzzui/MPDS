API Endpoints Available
GET    /api/v1/routines/           - Get all routines

DELETE /api/v1/routines/{id}       - Delete routine
POST   /api/v1/routines/{id}/activate - Activate routine

GET    /api/v1/routines/day/{day}  - Get routines by day

GET    /api/v1/routines/{id}/exercises - Get routine exercises
POST   /api/v1/routines/{id}/exercises - Add exercise to routine
